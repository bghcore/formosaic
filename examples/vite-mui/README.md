# Dynamic Forms — Vite + MUI Example

A lightweight Vite app demonstrating [@form-eng/core](https://www.npmjs.com/package/@form-eng/core) with the MUI adapter.

## Quick Start

```bash
cd examples/vite-mui
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the examples.

## Examples (tabs)

- **Basic** — Text, dropdown, toggle with validation
- **Business Rules** — Field dependencies and conditional visibility
- **Zod Schema** — Form generated from a Zod schema via `zodSchemaToFieldConfig()`
